## ❓ 常见问题 FAQ

### Q1：要全部学完才能开始毕业项目吗？
不需要。**01-04 + 06 必须**，05 / 07 / 08 可以边做项目边补。

### Q2：每天要花多少时间？
- 全职转行：6-8 小时/天，9 个月走完
- 在职学习：2-3 小时/天，12-15 个月走完
- 周末战士：1 天/周，2-3 年走完（可行但慢）

### Q3：可以跳过 Python，直接用 TypeScript 做后端吗？
可以。本路线推荐 Python（FastAPI）是因为 AI 生态最成熟。如果你只做 Web，Node + Hono / NestJS 也行。

### Q4：必须用 Claude 吗？换 GPT / Gemini / 开源模型行不行？
完全可以。本路线代码 90% 是模型无关的，换 API key 就好。Claude 是因为长文本 + 工具调用 + 编码能力最强而首选。

### Q5：我没有数学背景，能学 AI 吗？
能。本路线是 **AI 应用层**（用 API + RAG + Agent），不是 **AI 算法层**（炼模型）。前者只需高中数学。

### Q6：我电脑配置很差，能跑动吗？
能。本路线全部用云端 API + 云端部署，本地只跑 Node + Python，4GB 内存够用。

### Q7：学完能找到工作吗？
- 国内 AI 应用工程师：YES，有项目就行
- 国外远程：YES，但要求英语 + 作品集 + GitHub 活跃
- 大厂算法岗：本路线不直接对口，需要补研究背景

### Q8：毕业项目可以直接做副业赚钱吗？
完全可以。第 9 章就是教你这个。但平均独立开发者**第一年 MRR < $500**，请有合理预期。

### Q9：路线过时了怎么办？
本路线 6 个月会修订一次。AI 工具变化快，但**底层架构思想 5 年不会变**。学方法 > 学具体工具。

### Q10：学到一半想放弃怎么办？
正常。三个建议：
1. **降低强度**：从每天 3 小时降到 1 小时，不要断
2. **找学习同伴**：进 1 个本路线学习群（见 [[资源总览]]）
3. **去做项目**：理论卡住了就直接做，缺什么补什么
