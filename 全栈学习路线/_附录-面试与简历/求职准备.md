## 🎤 面试准备

### 八股文准备清单

#### 计算机基础（必备）
- TCP/UDP / HTTP/HTTPS / DNS
- HTTP 1/2/3 区别
- 进程 vs 线程 vs 协程
- 操作系统：内存管理 / 调度算法

#### Python
- GIL / asyncio 原理
- 装饰器 / 生成器 / 上下文管理器
- 内存管理（引用计数 + GC）
- 类相关：MRO / 元类 / 描述符

#### JavaScript
- 事件循环 / 微任务宏任务
- 闭包 / 原型链 / this
- Promise 实现
- V8 优化（Hidden Class / IC / TurboFan）

#### React
- Fiber / Reconciler / 调度器
- Hooks 实现原理
- 状态管理对比
- SSR vs CSR vs RSC

#### 数据库
- 索引（B+ 树）
- 事务隔离级别
- MVCC
- 主从复制 / 分库分表

#### AI / LLM
- Transformer 架构
- Attention 机制
- RAG 架构
- Agent 设计模式
- MCP 协议

### 算法（必刷）
- LeetCode Top 100：[Hot 100](https://leetcode.cn/studyplan/top-100-liked/)
- 剑指 Offer：[剑指 Offer 75](https://leetcode.cn/studyplan/coding-interviews-special/)
- 系统设计：[System Design Primer](https://github.com/donnemartin/system-design-primer)

### 项目讲解（最重要）

每个项目准备 3 个版本：
- **30 秒版**（电梯演讲）
- **5 分钟版**（面试常用）
- **30 分钟版**（深挖技术细节）

万能问题：
- 这个项目最难的地方是什么？怎么解决的？
- 如果 QPS 翻 10 倍怎么办？
- 怎么测试的？覆盖率多少？
- 上线后遇到什么问题？怎么排查的？
- 如果重做你会怎么改？
