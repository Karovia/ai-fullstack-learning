## 1. Python 是什么？为什么是它？🤔

**类比**：如果说编程语言是"工具"，那么：

- **C++** 像一台手动挡的跑车 — 速度快，但你得自己换挡、自己润滑（管理内存）。
- **JavaScript** 像一台浏览器里的电动车 — 上路方便，但只在浏览器里跑得最舒服。
- **Python** 像一台自动挡 SUV — 启动快、能拉货（数据分析）、能爬山（AI / 后端 / 脚本），还自带导航（丰富的标准库）。

| 语言 | 学习曲线 | 速度 | 主要场景 |
|------|---------|------|---------|
| Python | 平 | 慢 | AI、脚本、后端、数据 |
| JavaScript | 中 | 中 | 浏览器、网页、Node 服务 |
| C++ | 陡 | 极快 | 游戏引擎、操作系统、嵌入式 |

> 💡 **划重点**：Python 是 "**胶水语言**" — 它的强项不是单打独斗最快，而是把别人的工具（C 库、AI 模型、数据库）粘在一起。

## 2. 第一个程序：Hello, World 👋

打开终端，输入 `python3` 进入交互式环境，然后敲：

```python
print("Hello, World!")  # print 是"打印"的意思，把括号里的东西显示到屏幕
```

**运行结果**：
```
Hello, World!
```

**逐行拆解**：
- `print(...)` — 函数（就像微波炉，丢进去食材，出来熟食）。这里"丢进去"的是一段文字，"出来"的是屏幕上的字。
- `"Hello, World!"` — **字符串**，两边的双引号告诉 Python "这是一段文字，不是代码"。单引号 `'...'` 也行。
- `#` 之后的内容是 **注释**，Python 会忽略它，注释是写给"未来的你"看的。

## 3. 变量：贴标签的盒子 📦

**类比**：变量 = 一个**贴着标签的盒子**。盒子里能放东西，也能换东西，标签是你给它起的名字。

```python
name = "李雷"      # 把字符串"李雷"放进名为 name 的盒子
age = 18           # 把整数 18 放进名为 age 的盒子
age = age + 1      # 取出盒子里的值 + 1，再放回去 → 现在 age = 19
print(name, age)   # 打印两个盒子里的内容
```

**运行结果**：
```
李雷 19
```

> 💡 **划重点**：`=` 不是"等于"，而是"把右边塞进左边的盒子"。判断相等要用 `==`（双等号）。

### 命名规则
- 只能用字母、数字、下划线，不能用数字开头：`name1` ✅，`1name` ❌
- 不能用 Python 保留字：`if`、`for`、`class`…
- 推荐 **小写蛇形**：`user_name`，而不是 `userName`

## 4. 六大基本类型 🧱

| 类型 | 例子 | 含义 |
|------|------|------|
| `int` 整数 | `42`、`-7` | 没有小数点的数 |
| `float` 浮点数 | `3.14`、`-0.5` | 带小数点的数 |
| `str` 字符串 | `"你好"` | 一段文字 |
| `bool` 布尔 | `True`、`False` | 真 / 假（首字母大写！） |
| `NoneType` | `None` | "什么都没有"的占位符 |
| `bytes` 字节 | `b"abc"` | 计算机里最原始的二进制数据 |

```python
a = 42                  # int
b = 3.14                # float
c = "Python"            # str
d = True                # bool
e = None                # 没有值
f = b"hello"            # 字节串

print(type(a), type(b), type(c), type(d), type(e), type(f))
```

**运行结果**：
```
<class 'int'> <class 'float'> <class 'str'> <class 'bool'> <class 'NoneType'> <class 'bytes'>
```

> 💡 `type(x)` 用来查 x 是什么类型 — 它是程序员的"显微镜"。

## 5. 运算符大全 ➕➖✖️➗

```python
# 算术运算
print(7 + 3)    # 10
print(7 - 3)    # 4
print(7 * 3)    # 21
print(7 / 3)    # 2.333...   /  是普通除法，结果是 float
print(7 // 3)   # 2          // 是整除，丢弃小数
print(7 % 3)    # 1          %  是取余数
print(2 ** 10)  # 1024       ** 是幂运算

# 比较运算（结果一定是 bool）
print(3 > 2)    # True
print(3 == 3)   # True
print(3 != 4)   # True

# 逻辑运算
print(True and False)  # False  — 都真才真
print(True or False)   # True   — 一真就真
print(not True)        # False  — 取反

# 赋值简写
x = 10
x += 5   # 等价 x = x + 5
x *= 2   # 等价 x = x * 2
print(x) # 30

# 位运算（高级，先了解）
print(5 & 3)  # 1   按位与
print(5 | 3)  # 7   按位或
print(5 ^ 3)  # 6   异或
```

## 6. 字符串：最常用的数据 🔤

### 6.1 创建与拼接
```python
s1 = "Hello"
s2 = '世界'        # 单双引号都行
s3 = """支持
多行"""             # 三引号 = 多行字符串

print(s1 + ", " + s2)        # 拼接
print("ha" * 3)              # haha 重复 3 次 → hahahaha
```

### 6.2 切片（像切面包片）🍞
```python
s = "Python"
print(s[0])     # P  第一个字符（下标从 0 开始！）
print(s[-1])    # n  最后一个
print(s[0:3])   # Pyt 取 0、1、2，不含 3
print(s[2:])    # thon 从 2 到结尾
print(s[::-1])  # nohtyP 反转
```

### 6.3 f-string 格式化（强烈推荐）⭐
```python
name = "小明"
score = 95.5
print(f"{name} 这次考了 {score} 分，{'及格' if score >= 60 else '不及格'}")
```
**运行结果**：`小明 这次考了 95.5 分，及格`

### 6.4 10 个常用方法
```python
s = "  Hello Python  "
print(s.strip())          # 'Hello Python'  去首尾空格
print(s.lower())          # 全小写
print(s.upper())          # 全大写
print(s.replace("o","0")) # 替换
print(s.split())          # ['Hello', 'Python']  按空格分割
print("-".join(["a","b","c"]))  # a-b-c  用 - 连接
print("Python".startswith("Py"))# True
print("Python".endswith("on"))  # True
print("Python".find("th"))      # 2  找不到返回 -1
print(len("Python"))            # 6  长度
```

## 7. input() 与类型转换 ⌨️

```python
name = input("请输入姓名：")    # 程序停下来等你打字 + 回车
age = input("请输入年龄：")    # 注意：input 拿到的永远是 str！

print(type(age))               # <class 'str'>
age = int(age)                 # 显式转 int
print(f"明年你 {age + 1} 岁")
```

| 转换函数 | 用途 |
|---------|------|
| `int(x)` | 转整数 |
| `float(x)` | 转浮点 |
| `str(x)` | 转字符串 |
| `bool(x)` | 转布尔（0 / 空 → False） |

> 💡 **隐式转换**：`1 + 2.0` Python 自动把 1 升级成 `1.0`。
> **显式转换**：`int("3")` 你主动告诉 Python 要换。

## 8. 分支：if / elif / else 🚦

```python
score = int(input("分数："))
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 60:
    print("C")
else:
    print("不及格")

# 三元表达式
result = "及格" if score >= 60 else "不及格"
print(result)
```

> 💡 Python 用**缩进**（4 个空格）表示代码块，不像 C/Java 用 `{}`。**缩进错了就报错**。

## 9. 循环：while 与 for 🔁

### 9.1 while
```python
n = 1
while n <= 5:
    print(n)
    n += 1   # 千万别忘！否则死循环
```

### 9.2 for + range
```python
for i in range(5):       # 0,1,2,3,4
    print(i)

for i in range(1, 6):    # 1,2,3,4,5
    print(i)

for i in range(0, 10, 2):# 0,2,4,6,8  步长 2
    print(i)
```

### 9.3 break / continue
```python
for i in range(10):
    if i == 3:
        continue   # 跳过本次
    if i == 7:
        break      # 直接退出
    print(i)
# 输出 0 1 2 4 5 6
```

## 10. 实战 1：BMI 计算器 🏃

```python
height = float(input("身高（米）："))
weight = float(input("体重（千克）："))
bmi = weight / (height ** 2)

if bmi < 18.5:
    level = "偏瘦"
elif bmi < 24:
    level = "正常"
elif bmi < 28:
    level = "超重"
else:
    level = "肥胖"

print(f"你的 BMI 是 {bmi:.2f}，属于 {level}")
```

## 11. 实战 2：猜数字游戏 🎲

```python
import random
target = random.randint(1, 100)
count = 0
while True:
    guess = int(input("猜 1-100："))
    count += 1
    if guess < target:
        print("小了")
    elif guess > target:
        print("大了")
    else:
        print(f"猜对了！用了 {count} 次")
        break
```

## 🐛 常见错误集合

| 错误信息 | 原因 | 解决 |
|---------|------|------|
| `IndentationError` | 缩进不一致 | 全部用 4 个空格 |
| `NameError: x is not defined` | 用了没定义的变量 | 先 `x = ...` 再用 |
| `TypeError: can't add str and int` | 类型不匹配 | 用 `int()` / `str()` 转 |
| `ValueError: invalid literal for int()` | `int("abc")` | 检查输入是否是数字 |
| 死循环 | while 里忘了 `n += 1` | 加上更新语句 |
