# W03 · Python 函数与模块 🧩

> 函数是程序员最重要的工具 — 它把"会重复的逻辑"打包成"可复用的零件"。
> 模块是把这些零件**整理到不同抽屉**里的箱子。这一周，我们让代码从"一坨"变成"一组"。

---

## 1. 函数：可复用的微波炉 ♨️

**类比**：函数 = 微波炉。**输入**（食材）→ **加热**（处理）→ **输出**（熟食）。
你不需要每次都重新造一个微波炉，只要按按钮就好。

### 1.1 定义与调用
```python
def greet(name):              # def 定义，name 是参数
    return f"你好，{name}"     # return 把结果"端出来"

msg = greet("小明")            # 调用
print(msg)                    # 你好，小明
```

**逐行**：
- `def` 是关键字，告诉 Python "下面这块是个函数"
- `name` 是**形式参数**（站位的标签），调用时把"小明"塞进 `name`
- `return` 把结果交给外面；如果没有 `return`，函数返回 `None`

### 🎯 练习
写一个函数 `is_even(n)` 返回 n 是否偶数。

<details>
<summary>答案</summary>

```python
def is_even(n):
    return n % 2 == 0
print(is_even(4))   # True
```
</details>

## 2. 参数的 4 种形式 🎛️

```python
def order(name, size="中杯", *toppings, **options):
    print(f"{name} 一杯 {size}")
    print("加料：", toppings)
    print("配置：", options)

order("拿铁")
# 拿铁 一杯 中杯
# 加料：()
# 配置：{}

order("美式", "大杯", "糖浆", "冰块", sugar="少", milk="燕麦")
# 美式 一杯 大杯
# 加料：('糖浆', '冰块')
# 配置：{'sugar': '少', 'milk': '燕麦'}
```

| 形式 | 含义 |
|------|------|
| `name` | 位置参数（必填） |
| `size="中杯"` | 默认参数 |
| `*toppings` | 收集多余位置参数 → tuple |
| `**options` | 收集多余关键字参数 → dict |

> 💡 顺序口诀：**位置 → 默认 → \*args → \*\*kwargs**，颠倒报错。

### 关键字调用
```python
def power(base, exp):
    return base ** exp

print(power(2, 10))          # 1024
print(power(exp=10, base=2)) # 1024  通过名字传参，顺序无所谓
```

## 3. 作用域：LEGB 规则 🔭

**类比**：找变量像找钥匙 — 先找**自己抽屉**（Local），再找**爸妈抽屉**（Enclosing），再找**家里柜子**（Global），最后找**社区仓库**（Built-in）。

```
L (Local)      函数自己的局部变量
E (Enclosing)  外层函数的变量（嵌套函数）
G (Global)     模块全局
B (Built-in)   print / len / range 这些 Python 自带的
```

```python
x = "global"

def outer():
    x = "outer"
    def inner():
        # x = "inner"        # 取消注释会变 inner
        print(x)             # 找不到 inner，往上找 outer
    inner()
outer()                     # outer

# 修改全局：
count = 0
def incr():
    global count
    count += 1
incr()
print(count)                # 1
```

## 4. lambda 匿名函数 ⚡

```python
square = lambda x: x * x
print(square(5))    # 25

# 常和 sort/map/filter 配合
nums = [(1,2),(3,1),(2,5)]
nums.sort(key=lambda t: t[1])
print(nums)         # [(3,1),(1,2),(2,5)]
```

> 💡 lambda 只能写**一行表达式**，复杂逻辑用 def。

## 5. 高阶函数：map / filter / reduce 🚀

```python
nums = [1, 2, 3, 4, 5]

# map：对每个元素做处理
print(list(map(lambda x: x*x, nums)))   # [1, 4, 9, 16, 25]

# filter：过滤
print(list(filter(lambda x: x%2==0, nums)))   # [2, 4]

# reduce：累计
from functools import reduce
print(reduce(lambda a, b: a+b, nums))   # 15

# 但很多时候列表推导式更易读：
print([x*x for x in nums])
print([x for x in nums if x%2==0])
```

| 工具 | 等价推导式 | 何时用 |
|------|-----------|--------|
| `map(f, xs)` | `[f(x) for x in xs]` | 数据流式处理 |
| `filter(f, xs)` | `[x for x in xs if f(x)]` | 同上 |
| `reduce(f, xs)` | 需要 for 循环写 | 聚合（求和、求最大） |

## 6. 文档字符串 docstring 📝

```python
def divide(a, b):
    """两数相除。

    Args:
        a: 被除数
        b: 除数（不能为 0）

    Returns:
        商，类型 float
    """
    return a / b

print(divide.__doc__)
help(divide)   # 生成帮助文档
```

> 💡 写 docstring 是专业习惯。IDE 和 `help()` 都靠它。

## 7. 模块 import 的 4 种写法 📦

新建两个文件：

```python
# math_utils.py
PI = 3.14159
def add(a, b):
    return a + b
```

```python
# main.py
import math_utils                       # 写法 1：导入整个模块
print(math_utils.add(1, 2))

import math_utils as mu                 # 写法 2：起别名
print(mu.PI)

from math_utils import add              # 写法 3：导入单个
print(add(3, 4))

from math_utils import *                # 写法 4：全部导入（不推荐）
```

> 💡 不推荐 `from xxx import *` — 会污染命名空间，难追踪变量来自哪里。

## 10. 文件 IO：with open 📁

```python
# 写
with open("hello.txt", "w", encoding="utf-8") as f:
    f.write("第一行\n第二行\n")

# 读
with open("hello.txt", "r", encoding="utf-8") as f:
    content = f.read()           # 读全部
print(content)

# 按行读（推荐，省内存）
with open("hello.txt", encoding="utf-8") as f:
    for line in f:
        print(line.rstrip())
```

| 模式 | 含义 |
|------|------|
| `r` | 读（默认） |
| `w` | 写（**会清空原文件！**） |
| `a` | 追加 |
| `b` | 二进制（如 `rb`、`wb`） |
| `x` | 创建（已存在则报错） |

> 💡 `with` 自动关文件，不会忘 — 强烈推荐永远用 with。

## 11. pathlib：现代化路径处理 🛣️

```python
from pathlib import Path

p = Path("data") / "users.txt"       # 跨平台拼接
print(p)                             # data/users.txt

p.parent.mkdir(parents=True, exist_ok=True)
p.write_text("hello", encoding="utf-8")
print(p.read_text(encoding="utf-8")) # hello

print(p.exists(), p.is_file(), p.suffix, p.stem)
# True True .txt users

# 遍历目录
for f in Path(".").glob("*.py"):
    print(f.name, f.stat().st_size)
```

## 12. JSON 序列化与反序列化 🔄

**类比**：JSON 像快递员的"打包标准格式"。Python 对象太复杂，存盘 / 联网前需要"打包"成 JSON 字符串。

```python
import json

data = {"name": "李雷", "age": 18, "skills": ["py", "js"]}

# 对象 → 字符串
text = json.dumps(data, ensure_ascii=False, indent=2)
print(text)

# 字符串 → 对象
back = json.loads(text)
print(back["skills"][0])      # py

# 直接和文件打交道
with open("user.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

with open("user.json", encoding="utf-8") as f:
    obj = json.load(f)
print(obj)
```

| 方法 | 含义 |
|------|------|
| `dumps` | dump string — 转字符串 |
| `loads` | load string — 从字符串读 |
| `dump`  | dump → file |
| `load`  | load ← file |

## 13. 实战 1：成绩管理系统模块化 🎓

文件结构：
```
score_app/
├── __init__.py
├── storage.py        # 文件读写
└── analyzer.py       # 分析
main.py
```

```python
# score_app/storage.py
import json
from pathlib import Path

PATH = Path("scores.json")

def load():
    if not PATH.exists():
        return []
    return json.loads(PATH.read_text(encoding="utf-8"))

def save(data):
    PATH.write_text(json.dumps(data, ensure_ascii=False, indent=2), encoding="utf-8")
```

```python
# score_app/analyzer.py
def average(students, subject):
    return sum(s[subject] for s in students) / len(students)
```

```python
# main.py
from score_app import storage, analyzer

data = storage.load()
data.append({"name": "小明", "math": 90})
storage.save(data)

print(analyzer.average(data, "math"))
```

## 15. 实战 3：JSON 配置文件读写 ⚙️

```python
import json
from pathlib import Path

CONFIG = Path("config.json")

def load_config():
    if CONFIG.exists():
        return json.loads(CONFIG.read_text(encoding="utf-8"))
    return {"theme": "light", "lang": "zh"}

def save_config(cfg):
    CONFIG.write_text(json.dumps(cfg, ensure_ascii=False, indent=2), encoding="utf-8")

cfg = load_config()
cfg["theme"] = "dark"
save_config(cfg)
print("已保存", cfg)
```

## 🐛 常见错误集合

| 错误 | 原因 |
|------|------|
| `UnboundLocalError` | 函数里又读又写全局变量没声明 `global` |
| `ModuleNotFoundError` | 没装 / 没在虚拟环境 / 路径不对 |
| `JSONDecodeError` | JSON 格式错了（如尾逗号、单引号） |
| 默认参数 `=[]` 坑 | 用 `=None` 然后函数里设默认 |
| 文件中文乱码 | `open(..., encoding="utf-8")` |
