## 1. 列表 list：会变长的购物车 🛒

**类比**：list 就像超市购物车 — 你能往里**塞**商品、**拿走**商品、**调换**商品的位置，整个购物车能装很多东西。

### 1.1 创建与索引
```python
fruits = ["苹果", "香蕉", "橘子"]
print(fruits[0])    # 苹果   下标从 0 开始
print(fruits[-1])   # 橘子   倒数第一
print(len(fruits))  # 3      长度
```

### 1.2 切片（同字符串）
```python
nums = [10, 20, 30, 40, 50]
print(nums[1:4])    # [20, 30, 40]
print(nums[::-1])   # [50, 40, 30, 20, 10]  反转
```

### 1.3 增删改查
```python
fruits = ["苹果", "香蕉"]

# 增
fruits.append("橘子")          # 末尾追加 → ['苹果','香蕉','橘子']
fruits.insert(1, "葡萄")       # 在下标 1 插入 → ['苹果','葡萄','香蕉','橘子']
fruits.extend(["梨","桃"])     # 追加多个

# 删
fruits.remove("香蕉")          # 按值删除（删第一个匹配的）
last = fruits.pop()           # 弹出最后一个并返回
del fruits[0]                 # 按下标删

# 改
fruits[0] = "西瓜"

# 查
print("西瓜" in fruits)        # True / False
print(fruits.index("梨"))      # 返回下标
print(fruits.count("梨"))      # 出现次数
```

### 1.4 列表推导式（Pythonic 利器）⭐
```python
# 普通写法
squares = []
for i in range(1, 6):
    squares.append(i * i)
# 推导式写法（一行搞定）
squares = [i * i for i in range(1, 6)]
print(squares)   # [1, 4, 9, 16, 25]

# 带筛选
evens = [i for i in range(20) if i % 2 == 0]
print(evens)     # [0, 2, 4, ..., 18]

# 双层循环
pairs = [(x, y) for x in [1,2] for y in [3,4]]
print(pairs)     # [(1,3),(1,4),(2,3),(2,4)]
```

> 💡 **划重点**：列表推导式 = `[表达式 for 元素 in 可迭代对象 if 条件]`，比 for 循环更短更快。

## 2. 元组 tuple：一旦决定就不能改的清单 🔒

**类比**：tuple 就像**已经打印出来的发票** — 上面的内容不能再改了。

```python
point = (3, 4)
print(point[0], point[1])  # 3 4

# point[0] = 10   # ❌ 报错 TypeError

# 解包（很常用！）
x, y = point
print(x, y)        # 3 4

# 一行交换两个变量
a, b = 1, 2
a, b = b, a
print(a, b)        # 2 1
```

### namedtuple：给位置起名字
```python
from collections import namedtuple
Point = namedtuple("Point", ["x", "y"])
p = Point(3, 4)
print(p.x, p.y)    # 3 4  — 比 p[0] / p[1] 易读
```

> 💡 何时用 tuple？— **数据不变**且**位置有意义**（比如坐标、日期）的时候。

## 3. 字典 dict：键值对档案柜 🗄️

**类比**：dict 就像**字典书** — 通过"词条"（键）查"释义"（值），查找极快。

### 3.1 基本用法
```python
person = {"name": "李雷", "age": 18, "city": "北京"}

print(person["name"])              # 李雷
person["age"] = 19                 # 修改
person["job"] = "学生"             # 新增
del person["city"]                 # 删除

print("name" in person)            # True
```

### 3.2 安全的查找：get / setdefault
```python
score = {"小明": 90}

print(score["小红"])                  # ❌ KeyError
print(score.get("小红"))              # None  键不存在时返回 None
print(score.get("小红", 0))           # 0     给个默认值

# setdefault: 没有就设上，有就返回
score.setdefault("小红", 60)
print(score)                         # {'小明': 90, '小红': 60}
```

### 3.3 遍历
```python
person = {"name": "李雷", "age": 18}

for key in person:                   # 默认遍历键
    print(key)

for key, value in person.items():    # 同时拿键值
    print(key, "=", value)

print(list(person.keys()))           # ['name', 'age']
print(list(person.values()))         # ['李雷', 18]
```

### 3.4 字典推导式
```python
nums = [1, 2, 3, 4]
square_map = {n: n*n for n in nums}
print(square_map)   # {1:1, 2:4, 3:9, 4:16}
```

### 🎯 练习 2：词频统计
统计句子 `"the quick brown fox jumps over the lazy dog the end"` 里每个单词出现的次数。

<details>
<summary>答案</summary>

```python
sentence = "the quick brown fox jumps over the lazy dog the end"
freq = {}
for word in sentence.split():
    freq[word] = freq.get(word, 0) + 1
print(freq)
# {'the': 3, 'quick': 1, 'brown': 1, ...}
```
</details>

## 4. 集合 set：天生去重 🪣

**类比**：set 就像一个**没有重复客人的会员名单** — 同一个人加入多次只算一次。

```python
nums = [1, 1, 2, 3, 3, 4]
unique = set(nums)
print(unique)       # {1, 2, 3, 4}

a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b)        # 并集 {1, 2, 3, 4, 5, 6}
print(a & b)        # 交集 {3, 4}
print(a - b)        # 差集 {1, 2}
print(a ^ b)        # 对称差 {1, 2, 5, 6}

# 增删
a.add(10)
a.discard(1)        # 不存在不报错
a.remove(2)         # 不存在会报错
```

### frozenset：不可变集合
```python
fs = frozenset([1, 2, 3])
# fs.add(4)   # ❌ 不能改
# 可以作为 dict 的键、放进 set 里
```

## 5. 字符串高级：正则 re 入门 🔍

**类比**：正则就像**搜索通配符** — 你描述"我要找的样子"，Python 帮你找出所有匹配的部分。

```python
import re

text = "我的电话是 13800138000，他的电话是 13912345678"
phones = re.findall(r"1[3-9]\d{9}", text)
print(phones)   # ['13800138000', '13912345678']

# 常用元字符
# \d  数字  \w 字母数字下划线  \s 空白
# .   任意字符
# +   1 个或多个   *  0 个或多个   ? 0 或 1
# {n} 恰好 n 个    {n,m} n 到 m 个

# 替换
print(re.sub(r"\d", "*", "abc123"))   # abc***
```

## 7. 内存视角：可变 vs 不可变 🧠

**类比**：
- **不可变**（int/str/tuple）→ 像便利贴上写好的字，要改就**贴一张新的**。
- **可变**（list/dict/set）→ 像白板，**能擦掉重写**。

```python
# 不可变：改了 = 创建新对象
a = "hello"
print(id(a))     # 比如 4400000000
a = a + " world"
print(id(a))     # id 变了！其实是新对象

# 可变：原地修改，id 不变
lst = [1, 2, 3]
print(id(lst))   # 比如 4500000000
lst.append(4)
print(id(lst))   # 还是 4500000000

# 坑！默认参数是可变对象
def add(item, target=[]):     # ❌ 危险
    target.append(item)
    return target

print(add(1))   # [1]
print(add(2))   # [1, 2]  — 共享了同一个列表！
```

> 💡 **划重点**：函数默认参数永远不要写可变对象，用 `target=None` 然后在函数里判断。

## 8. 实战 1：词频统计（升级版）📊

```python
import re
from collections import Counter

text = """
Python is great Python is easy Python is fun
We love Python because Python makes life easy
"""

words = re.findall(r"\w+", text.lower())   # 全部小写并提取单词
counter = Counter(words)
print(counter.most_common(3))   # [('python', 4), ('is', 3), ('easy', 2)]
```

## 9. 实战 2：学生成绩管理 🎓

```python
students = []  # 每个元素是 dict

def add_student(name, math, en):
    students.append({"name": name, "math": math, "en": en})

add_student("小明", 90, 85)
add_student("小红", 78, 92)
add_student("小刚", 65, 70)

# 平均分
for s in students:
    s["avg"] = (s["math"] + s["en"]) / 2

# 按平均分排序
students.sort(key=lambda s: s["avg"], reverse=True)

for s in students:
    print(f"{s['name']:>4}  数学{s['math']}  英语{s['en']}  平均{s['avg']:.1f}")
```

## 10. 实战 3：ASCII 词云 ☁️

```python
from collections import Counter

text = "apple banana apple orange banana apple apple orange grape"
counter = Counter(text.split())

for word, n in counter.most_common():
    print(f"{word:<8} {'#' * n}  ({n})")
```

```

orange   ##  (2)
```

## 🐛 常见错误集合

| 错误 | 原因 |
|------|------|
| `KeyError` | 字典访问了不存在的键 → 用 `.get()` |
| `IndexError` | 列表越界 → 检查 `len()` |
| `TypeError: unhashable type: 'list'` | list 不能当 dict 的键 → 用 tuple |
| 默认参数共享 | 别用 `def f(x=[])` |
| 修改列表时遍历 | 在遍历中 `lst.remove()` 会跳元素，先复制再改 |
