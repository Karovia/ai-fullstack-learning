# W05 路由与表单

## 0. 本周目标

- React Router v7：让单页应用拥有"假装多页"的能力
- React Hook Form：高性能表单
- Zod：类型安全校验
- 周五一气呵成，搭一个登录 / 列表 / 详情 / 后台 / 富文本的 mini 管理系统

类比：
- 路由 = **网页大楼的电梯**——按按钮换楼层（页面），URL 是楼层号
- RHF = **不打扰演员的导演**——只在该开机时开机，平时不重渲组件
- Zod = **海关安检**——任何外部数据进系统前都过一遍

## 1. React Router v7 入门

```bash
pnpm add react-router-dom
```

> v7 是 v6 的延续（同包名），API 几乎一致；本文写法在 v6.4+ 都通用。

### 1.1 createBrowserRouter + RouterProvider

```tsx
// src/router.tsx
import { createBrowserRouter, RouterProvider } from 'react-router-dom'
import Home from './pages/Home'
import About from './pages/About'

export const router = createBrowserRouter([
  { path: '/', element: <Home /> },
  { path: '/about', element: <About /> },
  { path: '*', element: <h1>404</h1> },
])

// src/main.tsx
import { RouterProvider } from 'react-router-dom'
import { router } from './router'
createRoot(document.getElementById('root')!).render(<RouterProvider router={router} />)
```

类比：路由表是「**地铁路线图**」，每个 `path` 是「**站台**」，`element` 是「**站台风景**」。

### 1.2 Link / NavLink / useNavigate

```tsx
import { Link, NavLink, useNavigate } from 'react-router-dom'

<Link to="/about">关于</Link>

<NavLink to="/about" className={({ isActive }) => isActive ? 'active' : ''}>
  关于
</NavLink>

const nav = useNavigate()
<button onClick={() => nav('/users/1')}>看用户</button>
<button onClick={() => nav(-1)}>返回</button>
```

⚠️ 永远别用 `<a href>` 跳内部路由——会触发整页刷新。

### 1.3 嵌套路由 + Outlet

让多个页面共用一个布局：

```tsx
// src/layouts/AppLayout.tsx
import { Outlet, Link } from 'react-router-dom'
export default function AppLayout() {
  return (
    <>
      <nav><Link to="/">首页</Link> | <Link to="/users">用户</Link></nav>
      <main><Outlet /></main>   {/* 子路由内容渲染在这 */}
    </>
  )
}
```

```tsx
// src/router.tsx
export const router = createBrowserRouter([
  {
    path: '/',
    element: <AppLayout />,
    children: [
      { index: true, element: <Home /> },
      { path: 'users', element: <UserList /> },
      { path: 'users/:id', element: <UserDetail /> },
    ],
  },
])
```

类比：`Outlet` = 「**画框中间空着的位置**」——子路由渲到这里，外框（菜单、Header）保持不变。

### 1.4 useParams / useSearchParams

```tsx
// /users/:id
const { id } = useParams<{ id: string }>()

// /search?q=hi&page=2
const [sp, setSp] = useSearchParams()
const q = sp.get('q') ?? ''
const setQ = (v: string) => setSp(prev => { prev.set('q', v); return prev })
```

URL 状态比组件 state 牛在——可分享、可刷新、可后退。

### 1.5 loader / action（数据路由）

让 React Router 接管数据获取（v6.4+ 新功能）。

```tsx
// src/pages/UserDetail.tsx
import { useLoaderData, LoaderFunctionArgs } from 'react-router-dom'

export async function userLoader({ params }: LoaderFunctionArgs) {
  const r = await fetch(`/api/users/${params.id}`)
  if (!r.ok) throw new Response('Not Found', { status: 404 })
  return r.json()
}

export default function UserDetail() {
  const user = useLoaderData() as User
  return <h1>{user.name}</h1>
}
```

```tsx
{ path: 'users/:id', element: <UserDetail />, loader: userLoader }
```

类比：loader 是「**进站前先取好票**」——页面渲染时数据已经在了，不用再 useEffect 拉。

> 用 TanStack Query 的项目通常**不用** loader，仍走 useQuery。两套同时上会重复请求。本周建议熟悉 loader，但实战项目里挑一种用。

### 1.6 路由懒加载 + Suspense

按需加载页面 chunk：

```tsx
import { lazy, Suspense } from 'react'
const Admin = lazy(() => import('./pages/Admin'))

{
  path: 'admin',
  element: (
    <Suspense fallback={<p>加载中…</p>}>
      <Admin />
    </Suspense>
  ),
}
```

打包后 Admin 单独成一个 chunk，访问 /admin 才下载。

### 1.7 路由守卫（鉴权）

```tsx
// src/components/RequireAuth.tsx
import { Navigate, useLocation } from 'react-router-dom'
import { useAuth } from '../store/auth'

export function RequireAuth({ children }: { children: React.ReactNode }) {
  const user = useAuth(s => s.user)
  const loc = useLocation()
  if (!user) return <Navigate to="/login" state={{ from: loc }} replace />
  return <>{children}</>
}
```

```tsx
{ path: 'admin', element: <RequireAuth><Admin /></RequireAuth> }
```

登录成功后跳回原页：

```tsx
const loc = useLocation()
const from = (loc.state as any)?.from?.pathname || '/'
nav(from, { replace: true })
```

## 2. React Hook Form

```bash
pnpm add react-hook-form
```

为什么不用受控？
- 受控 = 每次 keystroke 整个表单组件重渲
- RHF = 用 ref 把输入框注册到内部，**只有提交或校验时才触发渲染**

### 2.1 useForm + register

```tsx
import { useForm } from 'react-hook-form'

interface FormValues { email: string; password: string }

export default function LoginForm() {
  const { register, handleSubmit, formState: { errors } } = useForm<FormValues>()

  const onSubmit = (data: FormValues) => console.log(data)

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input
        {...register('email', { required: '邮箱必填', pattern: { value: /^\S+@\S+$/, message: '格式错' } })}
        placeholder="邮箱"
      />
      {errors.email && <p style={{ color: 'red' }}>{errors.email.message}</p>}

      <input type="password" {...register('password', { required: '密码必填', minLength: { value: 6, message: '至少 6 位' } })} />
      {errors.password && <p style={{ color: 'red' }}>{errors.password.message}</p>}

      <button type="submit">登录</button>
    </form>
  )
}
```

`{...register('email')}` 会展开成 `name`、`onChange`、`onBlur`、`ref`——一行注册搞定。

### 2.2 Controller（适配受控组件）

第三方 UI（antd Select、MUI 等）不暴露 ref，要用 Controller：

```tsx
import { Controller } from 'react-hook-form'
import Select from 'react-select'

<Controller
  name="city"
  control={control}
  rules={{ required: true }}
  render={({ field }) => <Select {...field} options={cities} />}
/>
```

### 2.3 性能可视化

打开 React DevTools Profiler，输入 10 次：
- 受控版：每次 keystroke 整个表单组件重渲
- RHF 版：只有 input 自己内部 ref 更新，上层组件 0 次重渲

100+ 字段表单是天差地别。

## 3. Zod 校验

```bash
pnpm add zod @hookform/resolvers
```

### 3.1 Schema 定义

```tsx
// src/schemas/login.ts
import { z } from 'zod'

export const loginSchema = z.object({
  email: z.string().email({ message: '邮箱格式错误' }),
  password: z.string().min(6, { message: '密码至少 6 位' }),
  remember: z.boolean().optional(),
})

export type LoginInput = z.infer<typeof loginSchema>   // ✨ 自动推导 TS 类型
```

类比：Zod schema = 「**安检流程图**」+「**类型定义**」二合一。一份代码，运行时校验和编译时类型都解决了。

### 3.2 与 RHF 集成

```tsx
import { useForm } from 'react-hook-form'
import { zodResolver } from '@hookform/resolvers/zod'
import { loginSchema, LoginInput } from '../schemas/login'

const { register, handleSubmit, formState: { errors } } = useForm<LoginInput>({
  resolver: zodResolver(loginSchema),
})

return (
  <form onSubmit={handleSubmit((d) => console.log(d))}>
    <input {...register('email')} />
    {errors.email && <p>{errors.email.message}</p>}
    <input type="password" {...register('password')} />
    {errors.password && <p>{errors.password.message}</p>}
    <button>登录</button>
  </form>
)
```

校验规则**集中**在 schema，不再散落 register。

### 3.3 高级用法

```tsx
const registerSchema = z.object({
  password: z.string().min(8),
  confirm: z.string(),
}).refine(d => d.password === d.confirm, {
  message: '两次密码不一致',
  path: ['confirm'],
})
```

错误信息中文化全局：

```tsx
import { z } from 'zod'
z.setErrorMap((iss, ctx) => {
  if (iss.code === 'too_small' && iss.type === 'string')
    return { message: `至少 ${iss.minimum} 个字符` }
  return { message: ctx.defaultError }
})
```

## 4. 文件上传（含进度）

```tsx
// src/components/Upload.tsx
import { useState } from 'react'

export default function Upload() {
  const [progress, setProgress] = useState(0)
  const [url, setUrl] = useState('')

  const onChange = (e: React.ChangeEvent<HTMLInputElement>) => {
    const file = e.target.files?.[0]
    if (!file) return
    const xhr = new XMLHttpRequest()
    xhr.upload.onprogress = (ev) => {
      if (ev.lengthComputable) setProgress(Math.round(ev.loaded / ev.total * 100))
    }
    xhr.onload = () => setUrl(JSON.parse(xhr.responseText).url)
    xhr.open('POST', '/api/upload')
    const fd = new FormData(); fd.append('file', file)
    xhr.send(fd)
  }

  return (
    <div>
      <input type="file" onChange={onChange} />
      <progress value={progress} max={100} />
      {url && <img src={url} style={{ maxWidth: 200 }} />}
    </div>
  )
}
```

> fetch API 至今**不支持上传进度事件**，所以 XHR 还活着。或者用 `axios` 的 `onUploadProgress`。

## 5. 富文本：Tiptap 简介

```bash
pnpm add @tiptap/react @tiptap/pm @tiptap/starter-kit
```

```tsx
// src/components/Editor.tsx
import { useEditor, EditorContent } from '@tiptap/react'
import StarterKit from '@tiptap/starter-kit'

export default function Editor() {
  const editor = useEditor({
    extensions: [StarterKit],
    content: '<p>Hello <strong>world</strong></p>',
  })

  if (!editor) return null

  return (
    <div>
      <div>
        <button onClick={() => editor.chain().focus().toggleBold().run()}>B</button>
        <button onClick={() => editor.chain().focus().toggleItalic().run()}>I</button>
        <button onClick={() => editor.chain().focus().toggleHeading({ level: 2 }).run()}>H2</button>
      </div>
      <EditorContent editor={editor} style={{ border: '1px solid #ccc', padding: 8 }} />
    </div>
  )
}
```

类比：Tiptap = 「**Notion 同款引擎**」（基于 ProseMirror），可以做出协同编辑、自定义节点等高级功能。本周只用入门。

### 6.1 Auth store（Zustand）

```tsx
// src/store/auth.ts
import { create } from 'zustand'
import { persist } from 'zustand/middleware'

interface User { id: string; name: string }
interface State {
  user: User | null
  login: (name: string) => void
  logout: () => void
}

export const useAuth = create<State>()(persist(
  (set) => ({
    user: null,
    login: (name) => set({ user: { id: '1', name } }),
    logout: () => set({ user: null }),
  }),
  { name: 'auth' }
))
```

### 6.3 Login 页（RHF + Zod）

```tsx
// src/pages/Login.tsx
import { useForm } from 'react-hook-form'
import { zodResolver } from '@hookform/resolvers/zod'
import { useNavigate, useLocation } from 'react-router-dom'
import { z } from 'zod'
import { useAuth } from '../store/auth'

const schema = z.object({
  name: z.string().min(2, '至少 2 个字符'),
  password: z.string().min(6, '至少 6 位'),
})
type Input = z.infer<typeof schema>

export default function Login() {
  const { register, handleSubmit, formState: { errors, isSubmitting } } =
    useForm<Input>({ resolver: zodResolver(schema) })
  const login = useAuth(s => s.login)
  const nav = useNavigate()
  const loc = useLocation()
  const from = (loc.state as any)?.from?.pathname || '/'

  const onSubmit = async (d: Input) => {
    await new Promise(r => setTimeout(r, 500))
    login(d.name)
    nav(from, { replace: true })
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)} style={{ maxWidth: 320 }}>
      <input {...register('name')} placeholder="用户名" />
      {errors.name && <p>{errors.name.message}</p>}
      <input type="password" {...register('password')} placeholder="密码" />
      {errors.password && <p>{errors.password.message}</p>}
      <button disabled={isSubmitting}>{isSubmitting ? '登录中…' : '登录'}</button>
    </form>
  )
}
```

### 6.4 PostList（Query + 搜索 URL 状态）

```tsx
// src/pages/PostList.tsx
import { useSearchParams, Link } from 'react-router-dom'
import { useQuery } from '@tanstack/react-query'

export default function PostList() {
  const [sp, setSp] = useSearchParams()
  const q = sp.get('q') ?? ''
  const { data } = useQuery({
    queryKey: ['posts'],
    queryFn: () => fetch('https://jsonplaceholder.typicode.com/posts?_limit=20').then(r => r.json()),
  })

  const list = (data ?? []).filter((p: any) => p.title.includes(q))

  return (
    <>
      <input value={q} onChange={e => setSp({ q: e.target.value })} placeholder="搜索" />
      <ul>
        {list.map((p: any) => (
          <li key={p.id}><Link to={`/posts/${p.id}`}>{p.title}</Link></li>
        ))}
      </ul>
    </>
  )
}
```

### 6.5 PostEdit（Tiptap 富文本）

```tsx
// src/pages/PostEdit.tsx
import { useEditor, EditorContent } from '@tiptap/react'
import StarterKit from '@tiptap/starter-kit'
import { useNavigate, useParams } from 'react-router-dom'
import { useState } from 'react'

export default function PostEdit() {
  const { id } = useParams()
  const nav = useNavigate()
  const [title, setTitle] = useState('')
  const editor = useEditor({ extensions: [StarterKit], content: '<p>开始写...</p>' })

  const save = async () => {
    const html = editor?.getHTML()
    console.log({ id, title, html })
    // 真实项目：mutation 到后端
    nav(`/posts/${id}`)
  }

  return (
    <div>
      <input value={title} onChange={e => setTitle(e.target.value)} placeholder="标题" />
      <EditorContent editor={editor} style={{ border: '1px solid #ccc', minHeight: 200, padding: 8 }} />
      <button onClick={save}>保存</button>
    </div>
  )
}
```
