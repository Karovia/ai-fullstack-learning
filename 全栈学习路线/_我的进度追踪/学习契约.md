# 📜 我的学习契约

> 在你脑子最热血的时候签这份契约。撑不下去的时候来读它。

## 我的承诺

**我，[填入你的名字]，承诺**：

从 2026 年 _______ 月 _______ 日起，用 12-18 个月时间，完成 [[全栈学习路线/README]] 中全部 9 章学习。

## 每日承诺

- 工作日每天投入至少 **3 小时**学习
- 周末每天投入至少 **6 小时**
- 每天至少 **1 个 commit** 到 GitHub
- 每天写 **1 条**学习日志（在 Daily Note）

## 每周承诺

- GitHub 至少 **5 天**有真实 commit
- 写一份 [[周报模板|周报]]
- 学完 **1 个**章节小节 + 完成对应任务

## 每章承诺

- **必须完成章节项目**才算学完
- 写一篇 **≥ 2000 字**的章节复盘文章公开发布
- 把章节项目的代码 push 到 GitHub 公开仓库

### 惩罚

## 反馈系统

- **每天**：GitHub 绿点 + Daily Note

- **每章**：项目上线 + 复盘文章

## 应急预案

### 卡住怎么办
1. 先 Google + 官方文档 30 分钟
2. 还卡 → 问 Claude（只问方向，不要答案）
3. 还卡 → 跳过去做下一节，3 天后再回来
4. 还是不行 → 在掘金/Stack Overflow 提问

### 焦虑怎么办
1. 关闭社交媒体 24 小时
2. 跑步 30 分钟
3. 翻看自己第一周的代码（看进步）
4. 打开 [[周报模板|周报]] 看自己已完成的事

### 想放弃怎么办
1. 重读这份契约
2. 看自己当初写下的目标
3. 联系学习伙伴
4. 强制再坚持 1 周再决定
