## 🔑 一、Symbol：独一无二的"原始类型"

> 📖 **类比**：Symbol 像一把"指纹钥匙"——每把都不同，永远不会撞键。

```javascript
const a = Symbol('desc');
const b = Symbol('desc');
a === b;  // false（描述只是给人看，值绝对唯一）

// 用途 1：避免对象属性冲突
const KEY = Symbol('id');
user[KEY] = 42;  // 第三方库不会误改

// 用途 2：定义"内置协议"，比如让对象可迭代
const obj = {
  [Symbol.iterator]() {
    let i = 0;
    return {
      next: () => ({ value: i++, done: i > 3 })
    };
  }
};
[...obj];  // [0, 1, 2]
```

### Symbol.for / keyFor（全局注册表）

```javascript
const x = Symbol.for('shared');
const y = Symbol.for('shared');
x === y;  // true（同一个！）
```

## 🔄 二、迭代器协议 + for...of

任何对象只要实现了 `[Symbol.iterator]() => { next() }`，就能被 `for...of`、`...展开`、`解构` 使用。

```javascript
function range(start, end) {
  return {
    [Symbol.iterator]() {
      let i = start;
      return {
        next: () => i < end ? { value: i++, done: false } : { value: undefined, done: true }
      };
    }
  };
}

for (const n of range(1, 4)) console.log(n);  // 1 2 3
[...range(1, 4)];  // [1, 2, 3]
```

### for...of vs for...in

| 循环 | 遍历 | 适用 |
|---|---|---|
| `for...in` | **键名**（含原型链） | 对象 |
| `for...of` | **值**（须可迭代） | 数组、Set、Map、字符串 |

```javascript
const arr = ['a', 'b'];
for (const k in arr) console.log(k);  // 0 1
for (const v of arr) console.log(v);  // a b
```

## ⏸️ 三、Generator：可暂停的函数

```javascript
function* gen() {
  yield 1;
  yield 2;
  return 3;
}

const it = gen();
it.next();  // { value: 1, done: false }
it.next();  // { value: 2, done: false }
it.next();  // { value: 3, done: true }
```

> 📖 **类比**：普通函数是"一镜到底的电影"，Generator 是"看一段暂停一下"的连续剧。

### 用 Generator 自动管异步（async/await 的雏形）

```javascript
function* fetchUser() {
  const u = yield fetch('/api/user').then(r => r.json());
  const o = yield fetch(`/api/order?u=${u.id}`).then(r => r.json());
  return o;
}
```

### 双向通信：`it.next(value)` 把值传给 yield

```javascript
function* talk() {
  const name = yield '你叫啥？';
  yield `你好 ${name}`;
}
const t = talk();
t.next();          // { value: "你叫啥？" }
t.next('小明');    // { value: "你好 小明" }
```

## 🪝 四、Proxy + Reflect（Vue 3 的核心）

> 💡 **核心**：在对象的"读 / 写 / 删"操作前后插入自己的逻辑。

```javascript
const user = { name: '小明', age: 18 };

const proxy = new Proxy(user, {
  get(target, key, receiver) {
    console.log(`读取 ${key}`);
    return Reflect.get(target, key, receiver);
  },
  set(target, key, value, receiver) {
    console.log(`设置 ${key} = ${value}`);
    return Reflect.set(target, key, value, receiver);
  },
  deleteProperty(target, key) {
    console.log(`删除 ${key}`);
    return Reflect.deleteProperty(target, key);
  }
});

proxy.name;        // 读取 name
proxy.age = 20;    // 设置 age = 20
delete proxy.age;  // 删除 age
```

### 为什么 Vue 3 改用 Proxy？

| 方案 | 缺点 |
|---|---|
| Vue 2 `Object.defineProperty` | 必须遍历每个属性、新增属性不响应、数组要 hack |
| Vue 3 `Proxy` | 一层代理拦截所有操作、新增/删除自动响应、嵌套懒代理 |

### mini 响应式（Proxy 实现）

```javascript
let activeFn;
const targetMap = new WeakMap();

function track(target, key) {
  if (!activeFn) return;
  let depsMap = targetMap.get(target);
  if (!depsMap) targetMap.set(target, (depsMap = new Map()));
  let dep = depsMap.get(key);
  if (!dep) depsMap.set(key, (dep = new Set()));
  dep.add(activeFn);
}

function trigger(target, key) {
  targetMap.get(target)?.get(key)?.forEach(fn => fn());
}

function reactive(obj) {
  return new Proxy(obj, {
    get(t, k, r) { track(t, k); return Reflect.get(t, k, r); },
    set(t, k, v, r) { const ok = Reflect.set(t, k, v, r); trigger(t, k); return ok; }
  });
}

function effect(fn) { activeFn = fn; fn(); activeFn = null; }

// 用法
const state = reactive({ count: 0 });
effect(() => console.log('count =', state.count));  // count = 0
state.count++;  // 自动打印 count = 1 ✨
```

## 🗑️ 五、WeakMap / WeakSet / WeakRef

> 📖 **核心区别**：弱引用——key 没人用了，垃圾回收自动清掉。

```javascript
const wm = new WeakMap();
let user = { name: '小明' };
wm.set(user, '元数据');
user = null;  // user 对象可被 GC，wm 里的条目自动消失

// 限制：key 必须是对象、不能遍历、没有 size
```

**应用场景**：

- 缓存对象的"私有元数据"，不阻止 GC
- DOM 节点关联额外信息

| 类型 | 用途 |
|---|---|
| `Map` | 任意 key 的字典，可遍历 |
| `WeakMap` | 对象 key、弱引用、不可遍历 |
| `Set` | 唯一值集合 |
| `WeakSet` | 对象集合、弱引用 |

## 📦 六、Map vs Object

| 对比 | Map | Object |
|---|---|---|
| key 类型 | 任意（包括对象、函数） | 字符串 / Symbol |
| 顺序 | 插入顺序保证 | 数字 key 优先排序 |
| 大小 | `map.size` | `Object.keys(obj).length` |
| 遍历 | `for..of` 直接用 | 需要 keys/values/entries |
| 性能 | 频繁增删更优 | 静态结构更优 |

```javascript
const m = new Map();
m.set('a', 1).set({}, 2);
m.has('a');     // true
m.get('a');     // 1
for (const [k, v] of m) console.log(k, v);
```

```javascript

const port = config.port ?? 3000;

  #n = 0;

  get value() { return this.#n; }
```

## 🏋️ 九、综合实战：mini-lodash 10 个函数

```javascript
// 1. cloneDeep
function cloneDeep(v, seen = new WeakMap()) {
  if (v === null || typeof v !== 'object') return v;
  if (seen.has(v)) return seen.get(v);
  const out = Array.isArray(v) ? [] : {};
  seen.set(v, out);
  for (const k of Reflect.ownKeys(v)) out[k] = cloneDeep(v[k], seen);
  return out;
}

// 2. debounce / 3. throttle —— 见 W06

// 4. curry
function curry(fn) {
  return function curried(...args) {
    return args.length >= fn.length
      ? fn(...args)
      : (...rest) => curried(...args, ...rest);
  };
}

// 5. flatten
const flatten = (arr, d = 1) =>
  d > 0 ? arr.reduce((a, v) => a.concat(Array.isArray(v) ? flatten(v, d - 1) : v), []) : arr.slice();

// 6. get
function get(obj, path, def) {
  const keys = Array.isArray(path) ? path : path.split('.');
  let cur = obj;
  for (const k of keys) {
    if (cur == null) return def;
    cur = cur[k];
  }
  return cur === undefined ? def : cur;
}

// 7. set
function set(obj, path, val) {
  const keys = Array.isArray(path) ? path : path.split('.');
  keys.reduce((cur, k, i) => {
    if (i === keys.length - 1) cur[k] = val;
    else cur[k] = cur[k] ?? (typeof keys[i + 1] === 'number' ? [] : {});
    return cur[k];
  }, obj);
  return obj;
}

// 8. merge（深合并）
function merge(target, ...sources) {
  for (const s of sources) {
    for (const k of Object.keys(s)) {
      if (s[k] && typeof s[k] === 'object' && !Array.isArray(s[k])) {
        target[k] = merge(target[k] ?? {}, s[k]);
      } else target[k] = s[k];
    }
  }
  return target;
}

// 9. pick
const pick = (obj, keys) => keys.reduce((o, k) => (k in obj ? (o[k] = obj[k], o) : o), {});

// 10. omit
const omit = (obj, keys) => Object.fromEntries(Object.entries(obj).filter(([k]) => !keys.includes(k)));
```

## 🐛 常见错误

| 现象 | 原因 |
|---|---|
| Symbol 转字符串报错 | 必须显式 `String(sym)` 或 `sym.description` |
| Proxy 拦不住 `Reflect.get` 之外 | 内部方法走 [[OwnKeys]] 等，要按需实现 |
| WeakMap 不能 forEach | 设计如此，没有遍历 API |
| `??` vs `\|\|` 用错 | `0 \|\| 9 === 9`；`0 ?? 9 === 0` |
