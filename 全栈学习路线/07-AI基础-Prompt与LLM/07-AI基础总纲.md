## 📖 详细内容

### W1 - LLM 全景与 API
- 主流模型：Claude (Sonnet/Opus/Haiku) / GPT-4o / Gemini / Llama / DeepSeek / Qwen
- 闭源 vs 开源 vs 本地（Ollama）
- 价格 / 上下文窗口 / 多模态 / Tool Use 能力对比
- Anthropic API 完整教程（messages / streaming / system / extended thinking）
- 配额、速率限制、错误处理

### W2 - Prompt 进阶
- 6 大原则：清晰 / 上下文 / 角色 / 步骤 / 示例 / 输出格式
- Chain of Thought（CoT）+ Tree of Thought（ToT）
- Few-shot / Zero-shot
- Self-Consistency（多次采样投票）
- 结构化输出（JSON Mode / Tool Use 强制结构）
- 防注入与越狱
- **必读**：[Anthropic Prompt Engineering Course](https://github.com/anthropics/courses)

### W3 - Function Calling / Tool Use
- 工具定义（schema）
- 调用循环（while loop until done）
- 并行工具调用
- 错误处理 + 重试
- 工具结果反馈格式

### W4 - RAG 系统
- **数据准备**：解析 PDF/HTML/Markdown（Unstructured.io / Docling）
- **切片**：固定窗口 / 语义 / 递归字符 / 标题
- **向量化**：embedding 模型选择（text-embedding-3 / bge-m3 / Voyage）
- **存储**：pgvector / Qdrant / LanceDB
- **检索**：向量 + BM25 混合 + reranker（Cohere Rerank / bge-reranker）
- **生成**：拼 prompt + 引用（citations）
- **进阶**：HyDE / 父子文档 / GraphRAG

### W5 - 评测
- 离线评测数据集构建
- 自动指标：BLEU / ROUGE（局限性）
- LLM-as-a-Judge（让大模型当裁判）
- 人工评测（rubric + 平台 Argilla）
- 框架：[Promptfoo](https://promptfoo.dev/) / [Inspect](https://inspect.aisi.org.uk/) / [Ragas](https://github.com/explodinggradients/ragas)

### W6 - LLM 原理 + 从零实现 GPT
- Transformer 架构（Self-Attention / FFN / Residual / LayerNorm）
- Tokenizer（BPE）+ Embedding
- 训练流程（Pretrain → SFT → RLHF → DPO）
- 推理优化（KV Cache / Flash Attention / 量化）
- **跟着做 [LLMs-from-scratch (rasbt)](https://github.com/rasbt/LLMs-from-scratch) 全部章节**

## 🚀 章节大项目：`AskMyDocs` —— 企业级文档问答系统

### 需求
做一个能让企业上传内部文档（PDF/Word/Markdown/Confluence 导出），员工提问的问答系统：
- 多文档上传 + 自动解析（含表格 + 图片 OCR）
- 智能切片 + 元数据提取
- 混合检索（向量 + BM25 + 重排）
- 流式回答 + 引用原文（点击跳转 PDF 高亮）
- 多轮对话 + 对话历史
- 权限控制（不同部门看不同文档）
- 评测仪表盘（准确率 / 召回率 / 延迟）
- 反馈机制（👍/👎 收集）

### 技术栈
- 前端：Next.js 15 + shadcn/ui + Vercel AI SDK
- 后端：FastAPI + SQLAlchemy
- 数据库：PostgreSQL + pgvector + Redis
- LLM：Claude Sonnet（主）+ Haiku（路由/分类）
- Embedding：Voyage AI 或开源 bge-m3
- Reranker：Cohere Rerank 3
- 文档解析：Docling 或 Unstructured.io
- 评测：Ragas + Promptfoo
- 部署：Docker Compose

### 验收标准
- [ ] 上传 100 页 PDF，10 秒内完成索引
- [ ] 提问首字符响应 < 1.5s（流式）
- [ ] 答案准确率（人工评 100 题）≥ 85%
- [ ] 引用必须可点击跳转到原文
- [ ] 支持 5 种以上文档格式
- [ ] 评测仪表盘可用
- [ ] 完整 README + 演示视频
- [ ] 已部署上线，公网可访问

### 进阶挑战
- 加入 GraphRAG（知识图谱增强）
- 接入企业 SSO（Google/Microsoft）
- 加入 multi-modal（图片提问 → 答案）
- 用 [Letta](https://github.com/letta-ai/letta) 做长期记忆
